# Career Path AI

## AI-Powered Nigerian Career Guidance and Skill Roadmap Platform

## 1. Project Overview

**Career Path AI** is an AI-powered career planning and employability platform built for **Nigerian students, undergraduates, fresh graduates, NYSC members, and career switchers**. Its goal is to help users discover **realistic career paths**, understand the **skills they need**, and follow a **personalized roadmap** that fits the **Nigerian job market**, their **budget**, and their **learning timeline**.

Unlike generic career platforms that provide broad global advice, Career Path AI focuses on the **Nigerian context**. It combines **AI-powered career guidance**, **market-aware recommendations**, **skill-gap analysis**, **roadmap generation**, and **progress tracking** into one system that helps users move from **confusion to clarity** and from **interest to employability**.

---

# 2. Problem Statement

A large number of young Nigerians make important career decisions with little practical guidance. Many choose university courses because of family pressure, assumptions, or lack of information. Some spend years in school only to discover that the job opportunities in their field are limited or that they lack the digital skills employers actually demand. Fresh graduates often leave school without a clear career path, no portfolio, and no roadmap for becoming employable.

The major issues include:

* Students choosing the wrong course or career path without enough guidance
* Undergraduates not knowing how to make themselves job-ready before graduation
* Graduates lacking digital skills and practical direction
* Career guidance platforms being too generic and not tailored to Nigeria
* Lack of affordable, localized learning paths based on a user’s budget and situation
* No clear bridge between **career advice** and **actual execution**

Career Path AI is designed to solve this by giving users **Nigeria-specific career direction** and a **step-by-step plan** to move toward employability.

---

# 3. Vision

To become the go-to **AI career navigation platform for Nigerians**, helping students and young professionals make informed career decisions, build relevant skills, and become job-ready through practical, affordable, and localized roadmaps.

---

# 4. Core Value Proposition

Career Path AI helps users answer three important questions:

1. **What career path realistically fits me in Nigeria?**
2. **What skills am I missing to become employable in that path?**
3. **What exact step-by-step roadmap should I follow to get there?**

---

# 5. Primary Target Users

## Main Users for MVP

The MVP should focus on **Nigerian undergraduates and fresh graduates** because they are the easiest and strongest user segment to solve for first.

### Primary target groups:

* **Undergraduates** who want to become employable before graduation
* **Fresh graduates / NYSC members** trying to enter a career path
* **Students interested in tech or digital careers** but unsure where to start

## Secondary Users (future expansion)

* Secondary school students choosing university courses
* Career switchers moving from one field to another
* Early professionals trying to move into higher-paying digital roles

---

# 6. Core Product Concept

Career Path AI is not just a chatbot. It is a **career planning engine** made up of several connected systems:

1. **User profile and assessment system**
2. **Career recommendation engine**
3. **AI career coach / chatbot**
4. **Skill-gap analysis system**
5. **Visual roadmap generator**
6. **Progress tracking and milestone system**
7. **Learning resource recommendation engine**
8. **Nigerian market data layer**

Together, these systems help a user move from **profile → career recommendation → roadmap → progress tracking → job readiness**.

# 7.3 Career Recommendation Engine

This is the engine that decides which career paths are most suitable for the user.

## What it should do

After onboarding, the system should recommend **3–5 realistic career paths** for the user based on their background and goals.

## Example career paths

* Data Analyst
* Product Designer
* Frontend Developer
* Backend Developer
* UI/UX Designer
* Digital Marketer
* Product Manager
* Cybersecurity Analyst
* Customer Success Specialist
* Technical Writer
* Social Media Manager
* Virtual Assistant
* DevOps Engineer
* Business Analyst

## Factors used to rank career paths

* User’s education background
* Current course / discipline
* Interest areas
* Existing skills
* Budget
* timeline / urgency
* Nigeria job market demand
* entry barrier of the role
* remote work potential
* affordability of learning resources
* skill overlap with what the user already knows

## What the recommendation output should show

For each recommended career path, show:

* Career title
* Match score / fit score
* Why it fits the user
* Average salary range in Nigeria
* Demand level in Nigeria
* Remote work potential
* Estimated learning difficulty
* Estimated time to become job-ready
* Key required skills
* Recommended next action

# 7.4 AI Career Coach / Chatbot

This is the interactive assistant that allows users to ask career questions in natural language.

## What the chatbot should do

The AI coach should behave like a knowledgeable Nigerian career mentor. It should use the user’s profile data and app context to answer personalized questions.

## Example user questions

* “Is my course still useful in Nigeria?”
* “Can I move from microbiology to data analysis?”
* “What tech skill can I learn with ₦10,000 monthly budget?”
* “What job can I get in 4 months?”
* “What is the easiest tech role to start from my background?”
* “What should I learn first if I want to become a product designer?”
* “Can I work remotely from Ilorin or Ibadan in this role?”

## Chatbot responsibilities

* Answer Nigeria-specific career questions
* Explain career options in simple language
* Suggest next steps based on user profile
* Explain trade-offs between career paths
* Recommend learning strategies
* Motivate the user and keep them progressing
* Clarify roadmap steps
* Answer “why was this role recommended to me?”

## Important chatbot behavior

The chatbot should not just be a general-purpose AI assistant. It should have access to:

* user profile
* recommended career paths
* roadmap data
* progress data
* curated Nigerian job market context
* resource library

That makes its answers personalized and product-aware.

If a user selects a career path like **Data Analyst**, the app should show:

* Excel
* SQL
* Power BI
* Data cleaning
* Data storytelling
* Portfolio projects

### What the user already has

* Basic Excel
* Good communication
* Strong math background

### What the user is missing

* SQL
* Power BI

* dashboard building
* interview preparation

## Skill-gap output should include

* Current readiness score
* Skills already covered
* Skills missing
* Priority order of missing skills
* Estimated time to close the gap
* Suggested roadmap generated from the missing skills

# 7.6 Personalized Roadmap Generator

This is one of the biggest features in the product.

The roadmap generator creates a **step-by-step learning plan** for the user based on:

* selected career path
* skill gap
* budget
* available time
* learning pace
* academic level / NYSC stage
* device and internet constraints

## Roadmap output structure

The roadmap can be divided into:

* Month 1
* Month 2
* Month 3
* Month 4
* etc.

or into stages like:

* Foundation
* Core skills
* Portfolio / projects
* Job readiness

## Example roadmap for a Data Analyst user

### Month 1

* Learn Excel basics
* Learn data concepts
* Complete beginner spreadsheet exercises

### Month 2

* Learn SQL basics
* Practice SELECT, WHERE, GROUP BY
* Complete mini data-cleaning tasks

### Month 3

* Learn Power BI or Tableau
* Build first dashboard project
* Write project summary

### Month 4

* Build portfolio project
* Optimize LinkedIn / CV
* Prepare for entry-level data analyst applications

## Each roadmap step can contain:

* task title
* description
* estimated time
* difficulty level
* recommended learning resources
* deadline / milestone
* completion status

# 8.1 Step-by-Step User Flow

## Step 1: User creates account

The user signs up using email/password or Google login.

## Step 2: User completes onboarding

The app asks the user questions about:

* education level
* current course
* interests
* budget
* career goals
* device access
* timeline

## Step 3: Profile data is analyzed

The app sends the user profile to the **career recommendation engine**.

## Step 4: Career recommendations are generated

The system ranks several possible career paths and shows the top options.

## Step 5: User views recommended paths

The user sees details such as:

* why each path fits
* salary range
* skills needed
* estimated learning timeline

## Step 6: User selects one preferred career path

The user clicks a recommended path they want to pursue.

## Step 7: Skill-gap analysis is generated

The app compares:

* user’s current skills
* selected career path requirements

Then it shows:

* what the user has
* what the user lacks
* what to learn first

## Step 8: Roadmap is generated

A personalized roadmap is created with:

* stages
* monthly goals
* tasks
* resources
* milestones

## Step 9: User begins learning journey

The user starts completing tasks from the roadmap.

## Step 10: User interacts with AI career coach

The user can ask questions at any point:

* about the roadmap
* about career choices
* about learning strategies
* about role comparisons

## Step 11: User tracks progress

As tasks are completed, the dashboard updates:

* progress %
* completed milestones
* remaining steps

## Step 12: Roadmap adapts over time

If the user changes goals, pace, or circumstances, the app can regenerate or adjust the roadmap.

# 15. Recommendation Logic (High-Level)

The recommendation engine can score career paths using weighted criteria.

## Example factors

* Interest match → 25%
* Background / course relevance → 20%
* skill overlap → 15%
* budget compatibility → 10%
* timeline fit → 10%
* Nigeria demand → 10%
* remote work potential → 5%
* affordability of learning path → 5%

The system can assign a total score and rank the top career paths.
